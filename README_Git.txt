2 - Script pour cloner un repository GitHub en local.

-> Ouvrir PowerShell ISE
-> Fichier -> Ouvrir & Choisir le fichier "ScriptClone.ps1"
-> Avant de lancer le script, vérifier le chemin et modifier si besoin.
	Exemple : si vous souhaitez sauvegarder dans le dossier 4SRC dans C:, il faudra changer le chemin dans les ''
		cd 'C:\4SRC'
-> On peut maintenant lancer le script avec la touche F5.
